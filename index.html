Tutorial to Make a React Application

Create a New Project

  1. Create a new Folder
  2. Run npm init -yCopy in the folder

Install React and ReactDOM

In the directory run npm install react react-domCopy

Set up Babel

Install all necessary packages: npm install @babel/core @babel/preset-env @babel/preset-react babel-loaderCopy

Create a file called .babelrc and paste in: {"presets": ["@babel/preset-env", "@babel/preset-react"]}Copy

Set up Webpack

Install necessary packages: npm install --save-dev webpack webpack-cli webpack-dev-server style-loader css-loader babel-loader html-webpack-plugin mini-css-extract-pluginCopy

Create webpack.config.js with the provided configuration.

Add Scripts

In package.json, add:

"start": "webpack-dev-server --mode development --open", "build": "webpack --mode production"Copy

Setup for Development

Create a 'src' folder
and add index.html,
index.js,
style.css.
Create a 'public' folder as well.

index.js

Example content for index.js file.

add:

import React from 'react';

import ReactDOM from 'react-dom';

import "./style.css";

// Define a simple React component

const App = () => {

return <h1>This is a test</h1>;

};

// Render the App component into a div with the id of 'root'

ReactDOM.render(<App />, document.getElementById('root'));
